My UX skills:

User Research

The first stage of conducting user research involves brainstorming. This step clearly defines the problem, any assumptions and insights the researchers have. We generate hypothesis, user insight, and problem statements, and proto-personas.

User empathy

We capture user empathy by generating affinity diagrams (incl. POV statements), empathy maps, and user-personas.

Qualitative interviews

A interview plan is generated with questions broken up into sections that explore all aspects of the concept we want to develop into a product. Questions are curated to explore users pinpoints and desires. To ensure that the user answers questions genuinely, we limit the number and duration of the interviews.
